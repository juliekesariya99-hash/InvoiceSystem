# Invoice Management System

A professional Django-based Invoice Management System for managing customers, products, invoices, PDF generation, QR codes, and dashboard analytics.

## Features

* Customer Management
* Product Management
* Invoice Creation
* Invoice Detail View
* PDF Invoice Download
* QR Code Generation
* Dashboard Analytics
* Django Admin Panel

## Technologies Used

* Python
* Django
* SQLite
* Bootstrap 5
* ReportLab
* qrcode

## Project Structure

```text
InvoiceSystem/
├── billing/

├── manage.py
├── README.md
└── .gitignore
```

## Installation

```bash
git clone https://github.com/juliekesariya99-hash/InvoiceSystem.git
cd InvoiceSystem
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
